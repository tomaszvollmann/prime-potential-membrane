# REFERENCE_IMPLEMENTATION

Prime Potential Membrane (PPM)  
Reference Implementation — Generator v0.7  
Freeze v1.0 — Vienna, Austria — January 2026

---

## 1. Purpose

The PPM Generator (v0.7) is a reference implementation of the Prime Potential
Membrane (PPM) standard.

It demonstrates one possible method of constructing a valid PPM deployment
using structured JSON modules under:

```txt
/.well-known/membrane.json
/.well-known/membrane/
```

The generator exists to make the structural logic of PPM visible and operational.

It is not required for PPM implementation.

---

## 2. Relationship to the PPM Standard

Prime Potential Membrane (PPM) exists as:

- A decentralized structural standard  
- A conceptual trans-medial artwork  
- A declarative encounter model  

The generator is a tool.

The standard does not depend on the generator.

A valid PPM membrane may be constructed:

- Manually  
- Through custom software  
- Via AI-assisted generation  
- Through alternative implementations  

The PPM Generator is one reference implementation among potentially many.

---

## 3. Non-Dependency Principle

PPM-compliance is determined exclusively by adherence to the PPM specification
(**SPEC v1.0**) and its normative invariants.

Use of the generator is not required for conformance.

Absence of the generator does not invalidate a membrane.

The generator does not define the standard.

---

## 4. Status

The PPM Generator is a prototype implementation corresponding to Freeze v1.0.

Future versions MAY evolve independently without altering the canonical v1.0
specification unless explicitly stated.
